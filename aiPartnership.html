Case Study: Building PageNotes

PageNotes is a one-of-a-kind pixel-specific browser annotation system where notes 'stick' directly to the exact spot on any webpage you're viewing. The concept was born purely from my own vision, but brought to life with AI assistance—not AI domination.

I created the icons and iterated with Copilot and Claude to nail down all the fine details. The challenge wasn't just building a browser extension—it was creating persistent note positioning that survives page scrolling, resizing, and dynamic content changes.

  • My contribution: Original concept, UX design, icon creation, problem-solving
  • AI contribution: Chrome Extension architecture guidance, storage API implementation, cross-browser compatibility debugging

I didn't need to spend months learning every Chrome API detail to build something functional and unique. AI helped me implement the technical foundation in record time, freeing me to focus on perfecting the user experience and persistent 'sticking' behavior that makes PageNotes special.

The result? A fully functional, polished extension with features I dreamed up, executed faster than traditional solo development—without sacrificing my learning or creative ownership.
